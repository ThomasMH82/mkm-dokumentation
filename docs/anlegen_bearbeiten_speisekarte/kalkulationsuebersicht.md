# Kalkulationsübersicht anzeigen

**Anzeigen der Kalkulationsübersicht**


![speisenkarte_anlegen](/img/anlegen_bearbeiten_speisekarte/anlegen_bearbiten_speisekarte_2.png)

1. Klicken Sie in der Speisekarte auf den Button **„Kalkulation anzeigen“**
2. In dem Pop-Up Fenster, welches sich nun öffnet, sieht man eine **Übersicht aller Rezepturen**, die in dieser Speisekarte für die aktuelle Woche
eingetragen sind. Hier sieht man auf einen Blick die **geplante Anzahl an Portionen**, den **Wareneinsatz pro Portion**, den **Verkaufspreis pro
Portion** etc. d.h. alle Zahlen auf einen Blick, um zu erkennen ob die aktuelle Woche wirtschaftlich erfolgreich ist oder nicht.
