# Masterspeisekarten anlegen

![speisenkarte_anlegen](/img/anlegen_bearbeiten_speisekarte/anlegen_bearbiten_speisekarte_7.png)

Punkt 1 und 2 kennen Sie bereits von den herkömmlichen Speisekarten. Haken Sie nun Masterspeisekarte (Punkt 3) an. Bei Punkt 4 erscheinen nun
Menülinien, die im Vorfeld einmalig angelegt werden müssen. Hierzu findest Du [**hier**](/docs/anlegen_bearbeiten_speisekarte/speisekarten_verteilen#variante_1) mehr. Die Zielunternehmen werden angezeigt, sofern Sie den
Unternehmen zugewiesen wurden. Darüber hinaus können Sie noch Gruppen definieren. Dazu mehr [**hier**](/docs/anlegen_bearbeiten_speisekarte/speisekarten_verteilen#variante_2).

![speisenkarte_anlegen](/img/anlegen_bearbeiten_speisekarte/anlegen_bearbiten_speisekarte_8.png)

1. Klicken Sie oben in dem dunkelgrauen Kasten auf **Speisekarte** und im Anschluss auf **Menülinie**
2. Im Anschluss öffnet sich ein neues Fenster, in dem die Menülinien angelegt werden können. Sofern Sie bereits Menülinien angelegt haben, ist
diese Liste bereits ausgefüllt. Hierüber können Sie auch Menülinien im Anschluss bearbeiten
3. Nachdem Sie auf **Anlegen** geklickt haben, öffnet sich das letzte Fenster. Hier geben Sie den **Namen der Menülinie** ein. Unter „DisplayName“
können Sie einen anderen Namen vergeben, der bei dem Kunden auf der Speiskarte angezeigt werden soll.

![speisenkarte_anlegen](/img/anlegen_bearbeiten_speisekarte/anlegen_bearbiten_speisekarte_9.png)

1. Klicken Sie oben in dem dunkelgrauen Kasten auf Konfiguration und im Anschluss auf **Ziel-Unternehmen-Listen**
2. Um **Gruppen** anzulegen, klicken Sie hierfür auf **Anlegen**. Wie in dem o.g. Beispiel gibt es eine Gruppe für Schulen und eine für Betriebe. Sie können
dieser Gruppe beliebig viele Unternehmen hinzufügen, sofern diese ebenfalls Ihnen zugewiesen sind
3. Tragen Sie in dem **freien Kästchen** Ihre Gruppe ein und unter **Eintrag hinzufügen** können Sie die Unternehmen zuordnen
