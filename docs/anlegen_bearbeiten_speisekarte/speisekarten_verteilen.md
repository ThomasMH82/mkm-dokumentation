# Speisekarten verteilen

**Verteilung von Masterspeiseplänen**

## Variante 1 {#variante_1}

![speisenkarte_anlegen](/img/anlegen_bearbeiten_speisekarte/anlegen_bearbiten_speisekarte_10.png)

Speisekarten können über zwei Wege verteilt werden:
1. Über die Speisekartenansicht können Sie links auf das **leere Kästchen**, im Anschluss auf **Aktion** und zum Schluss auf **Menüplan-Synchronisation** klicken
2. Als nächstes öffnet sich folgendes Fenster. Hier stehen nochmals die ausgewählten Betriebe und ggf. Produkte bzw. Rezepturen (2. Absatz), die im
Zielunternehmen nicht vorhanden sind. **Voraussetzung für die Verteilung sind gleiche Stammdaten**.
3. Sollten Rezepturen oder Produkte fehlen, stellen Sie sicher, dass diese vorher gepflegt werden. Sobald Sie auf **Anlegen** klicken, **wird die
Speisekarte verteilt**. Dies kann einige Sekunden dauern.

## Variante 2 {#variante_2}

![speisenkarte_anlegen](/img/anlegen_bearbeiten_speisekarte/anlegen_bearbiten_speisekarte_11.png)

1. Sie können ebenfalls über die **Speisekarten Bearbeitungsansicht** die Speisekarte verteilen. Hierzu stellen Sie sich in die jeweilige Speisekarte
und klicken auf **Aktion** und im Anschluss auf **Menüplan-Synchronisation**
2. Im Anschluss öffnet sich wie bei Variante das gleiche Fenster. Sobald Sie auf **Anlegen** klicken, **wird die Speisekarte verteilt**.
Dies kann einige Sekunden dauern.
