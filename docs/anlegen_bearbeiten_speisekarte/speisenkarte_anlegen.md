# Speisenkarte anlegen

**Speisenkarte anlegen**

![speisenkarte_anlegen](/img/anlegen_bearbeiten_speisekarte/anlegen_bearbiten_speisekarte_1.png)

1. In das Speisekartenmodul gelangen Sie entweder über die **linke Menüleiste** oder über das **Dashboard**.
2. Im Speisekartenmodul angekommen, können Sie über den Button **„Anlegen“** eine neue Speisekarte anlegen mit Klick in Ihre bestehenden
Speisekarten diese bearbeiten.

![speisenkarte_anlegen](/img/anlegen_bearbeiten_speisekarte/anlegen_bearbiten_speisekarte_2.png)

1. Wenn Sie eine neue Speisekarte anlegen, geben Sie zuerst einen **Namen** ein und wählen im Anschluss (2.) das **Anfangsdatum** sowie die **geplante
Anzahl der Wochen** aus.
3. Nun tragen Sie noch ihre gewünschten **Menülinien** ein und (4.) wählen **die Service-Tage** aus.
5. Um in den **Bearbeitungsmodus** der Speisekarte zu gelangen, klicken Sie auf **„Speisekarte anzeigen“**
