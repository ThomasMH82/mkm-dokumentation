# Speisenkarte anlegen

**Speisenkarte anlegen**

![speisenkarte_anlegen](/img/anlegen_bearbeiten_speisekarte/anlegen_bearbiten_speisekarte_1.png)

1. In das Speisekartenmodul gelangen Sie entweder über die **linke Menüleiste** oder über das **Dashboard**.
2. Im Speisekartenmodul angekommen, können Sie über den Button **„Anlegen“** eine neue Speisekarte anlegen mit Klick in Ihre bestehenden
Speisekarten diese bearbeiten.

![speisenkarte_anlegen](/img/anlegen_bearbeiten_speisekarte/anlegen_bearbiten_speisekarte_2.png)

1. Wenn Sie eine neue Speisekarte anlegen, geben Sie zuerst einen **Namen (1)** ein und wählen im Anschluss das **Anfangsdatum (2)** sowie die **geplante Anzahl der Wochen** aus. 
2. Nun tragen Sie noch ihre gewünschten **Menülinien (3)** ein und wählen aus welche dieser Menülinien auch in dem **Ausdruck der Speisekarte (4)** erscheinen soll. Daraufhin wählt man noch die gewünschten **Servicetage (5)** aus. 
3. Um in den Bearbeitungsmodus der Speisekarte zu gelangen, klicken Sie auf **„Speisekarte anzeigen“ (6)**
