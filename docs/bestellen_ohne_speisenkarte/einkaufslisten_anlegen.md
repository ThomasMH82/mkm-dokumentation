# Einkaufslisten anlegen

**Anlegen von Einkaufslisten**

![bestellen_ohne_speisenkarte](/img/bestellen_ohne_speisekarte/bestellen_ohne_speisekarte_5.png)

Um eine Einkaufsliste anzulegen, muss das Modul **„Lieferantenbestellungen“** über das Dashboard oder das Icon in der Navigationsleiste auf der
linken Seite geöffnet werden. In dem Einkaufsmodul, welches sich nun öffnet, wählt man in der oberen linken Ecke den Button **„Bestellvorschau“**
aus.

![bestellen_ohne_speisenkarte](/img/bestellen_ohne_speisekarte/bestellen_ohne_speisekarte_6.png)

Wählen Sie den Reiter **„Vordefinierte Einkaufsliste“** aus.
Im anschließenden Auswahlfeld haben Sie **zwei Möglichkeiten**:
- Wählen Sie eine **bestehende Einkaufsliste (1)** aus, oder
- Legen Sie eine **neue Liste** an, indem Sie den gewünschten Namen eingeben und auf **„Lege … an“** klicken.

Nach dem Anlegen wird die neue Liste automatisch geöffnet. Haben Sie eine bestehende Liste ausgewählt, können Sie unter (2) die Liste bearbeiten.
Über den Button **„Eintrag hinzufügen“** können Sie nun die **Produkte auswählen**, die in der Einkaufsliste enthalten sein sollen.
Zum Abschluss klicken Sie auf **„Speichern“**, um die Liste dauerhaft zu speichern.
