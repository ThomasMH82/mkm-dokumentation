# Einkaufslisten bestellen

**Bestellen von Einkaufslisten**

![bestellen_ohne_speisenkarte](/img/bestellen_ohne_speisekarte/bestellen_ohne_speisekarte_7.png)

Um eine Einkaufsliste zu bestellen, gehen Sie wie folgt vor:
1. Wählen Sie das gewünschte **Lieferdatum** im entsprechenden Feld aus
2. Tragen Sie die **gewünschte Bestellmenge** in der Spalte **„Bestellmenge (EKE)“** für jedes Produkt ein
3. Klicken Sie anschließend auf **„Jetzt bestellen“**, um die Bestellung abzuschließen.
