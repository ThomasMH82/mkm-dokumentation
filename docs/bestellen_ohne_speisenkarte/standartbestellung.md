# Standardbestellung

**Standardbestellung**

![bestellen_ohne_speisenkarte](/img/bestellen_ohne_speisekarte/bestellen_ohne_speisekarte_1.png)

Um eine Standardbestellung auszulösen, muss das Modul **„Lieferantenbestellungen“** über das Dashboard oder das Icon in der Navigationsleiste auf
der linken Seite geöffnet werden. In dem Einkaufsmodul, welches sich nun öffnet, wählt man in der oberen linken Ecke den Button **„Anlegen“** aus, um
eine neue Bestellung anzulegen.

![bestellen_ohne_speisenkarte](/img/bestellen_ohne_speisekarte/bestellen_ohne_speisekarte_3.png)

In dem Fenster, das sich nun geöffnet hat, wird als erstes ein Lieferant ausgewählt. Daraufhin kann man im unteren Bereich des Fensters einzelne
Produkte des ausgewählten Lieferanten zu der Bestellung hinzufügen und die gewünschte Menge eintragen.

![bestellen_ohne_speisenkarte](/img/bestellen_ohne_speisekarte/bestellen_ohne_speisekarte_4.png)

Zuletzt wählt man noch das gewünschte Lieferdatum aus. Um die Bestellung dann tatsächlich zu versenden, muss man **„Auftrag bestätigen“**
auswählen.
