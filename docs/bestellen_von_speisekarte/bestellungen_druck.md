# Bestellungen drucken

**Druck von Bestellungen**

![speisekarte_bestellen](/img/bestellen_speisekarte/bestellen_mit_speisekarte_10.png)

Möchten Sie Ihre Bestellung, die automatisch an den Lieferanten übermittelt wurde, für Ihre Dokumentation drucken, so klicken Sie auf den Button
„Drucken“ und im Anschluss auf „Lieferantenbestellung“. Die erzeugte PDF können Sie nun herunterladen.
