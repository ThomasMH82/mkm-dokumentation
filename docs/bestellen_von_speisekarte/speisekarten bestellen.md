# Speisekarten bestellen

**Speisenkarten bestellen**

![speisekarte\_bestellen](/img/bestellen_speisekarte/bestellen_mit_speisekarte_1.png)

Sie gelangen entweder über die **linke Menüleiste** (1) oder über das **Dashboard** (2) in das Speisekartenmenü.

![speisekarte\_bestellen](/img/bestellen_speisekarte/bestellen_mit_speisekarte_2.png)

1. Zuerst werden die Portionen je Gericht gepflegt. Hierzu auf den **Button „Anzahl Portionen bearbeiten“** klicken.
2. Im Anschluss kann die Speisekarte bestellt werden

![speisekarte\_bestellen](/img/bestellen_speisekarte/bestellen_mit_speisekarte_3.png)

1. Im ersten Schritt tragen Sie nun alle **Portionsmengen** je Rezeptur/ je Tag ein. Die Ansicht ist bereits nach dem Servicetag gruppiert
2. Speichern Sie Ihre Änderungen und klicken dann auf den Button **„Speisekarte anzeigen“**. Im Anschluss gelangen Sie wieder zurück zu Ihrer Speisekarte.

![speisekarte\_bestellen](/img/bestellen_speisekarte/bestellen_mit_speisekarte_4.png)

Ihre Speisekarte enthält nun alle **notwendigen Portionen** (siehe Zahl neben Menschen-Symbol). Klicken Sie nun auf den Button **„Speisekarte bestellen“**. Sollten Sie keine Mengen eingetragen haben, weil Sie diese Rezeptur nicht bestellen möchten oder vergessen haben, erscheint diesbezüglich eine Meldung. Bestätigen Sie diese, oder brechen Sie diese ab, um fortzufahren.

![speisekarte\_bestellen](/img/bestellen_warenkorb_speisekarte_1.png)

1. **Wählen Sie zuerst alle Tage aus**, die Sie bestellen möchten
2. Zusätzlich besteht die Möglichkeit, **bereits vorhandene Ware** aus dem **Lager** zu verwenden, ohne diese im Rahmen einer neuen Bestellung mit aufnehmen zu müssen. Hierfür können Sie den mittleren Schalter je Tag anklicken und dadurch die Funktion abwählen.
3. Daraufhin klicken Sie auf "Warenkorb erstellen", woraufhin folgendes Fenster erscheint:

   ![](/img/bestellen_lieferdatum_speisekarte.png)
4. Hier sehen sie nun, falls es beim Onboarding angegeben wurde, die für Ihren Betrieb regulären Liefertage der in diesem Warenkorb betroffenen Lieferanten und wählen in der Spalte "Lieferdatum" noch ein gewünschtes Lieferdatum pro Lieferant aus.
5. Daraufhin klicken Sie auf "Bestätigen" womit die Warenkörbe erzeugt wurden.
