# Speisekarten bestellen

**Speisenkarten bestellen**

![speisekarte_bestellen](/img/bestellen_speisekarte/bestellen_mit_speisekarte_1.png)

Sie gelangen entweder über die **linke Menüleiste** (1) oder über das **Dashboard** (2) in das Speisekartenmenü. 

![speisekarte_bestellen](/img/bestellen_speisekarte/bestellen_mit_speisekarte_2.png)

1. Zuerst werden die Portionen je Gericht gepflegt. Hierzu auf den **Button „Anzahl Portionen bearbeiten“** klicken.
2. Im Anschluss kann die Speisekarte bestellt werden

![speisekarte_bestellen](/img/bestellen_speisekarte/bestellen_mit_speisekarte_3.png)

1. Im ersten Schritt tragen Sie nun alle **Portionsmengen** je Rezeptur/ je Tag ein. Die Ansicht ist bereits nach dem Servicetag gruppiert
2. Speichern Sie Ihre Änderungen und klicken dann auf den Button **„Speisekarte anzeigen“**.
Im Anschluss gelangen Sie wieder zurück zu Ihrer Speisekarte. 

![speisekarte_bestellen](/img/bestellen_speisekarte/bestellen_mit_speisekarte_4.png)

Ihre Speisekarte enthält nun alle **notwendigen Portionen** (siehe Zahl neben Menschen-Symbol).
Klicken Sie nun auf den Button **„Speisekarte bestellen“**. Sollten Sie keine Mengen eingetragen haben, weil Sie diese Rezeptur nicht bestellen
möchten oder vergessen haben, erscheint diesbezüglich eine Meldung. Bestätigen Sie diese, oder brechen Sie diese ab, um fortzufahren. 

![speisekarte_bestellen](/img/bestellen_speisekarte/bestellen_mit_speisekarte_5.png)

1. Wählen Sie zuerst Ihr gewünschtes **Lieferdatum** aus
2. Nun **wählen Sie alle Tage aus**, die Sie bestellen möchten
3. Zusätzlich besteht die Möglichkeit, **bereits vorhandene Ware** aus dem **Lager** zu verwenden, ohne diese im Rahmen einer neuen Bestellung mit
aufnehmen zu müssen. Hierfür können Sie den mittleren Schalter je Tag anklicken und dadurch die Funktion abwählen.
