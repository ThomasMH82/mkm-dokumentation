# Status einsehen

**Status einsehen**

![speisekarte_bestellen](/img/bestellen_speisekarte/bestellen_mit_speisekarte_6.png)

:::tip **ACHTUNG**
Diese ändern sich je nach Zustand des Eintrages. 
(Siehe Montag)

:::
:::info **Mögliche Status**
- Im Warenkorb: Nach Speiseplanbestellung
- Bereit zur Produktion: Nach Auslösen der Bestellung beim Lieferanten und Warenannahme
- Gefertigt: Sobald die Bestellung produziert wurde
- Verkäufe erfasst: Nachdem alle gefertigten Rezepturen über den MKM verkauft wurden
- Übersprungen: Sofern ein Eintrag mit einer Menge von 0 versehen und bestellt wurde
:::
Diese Produkte wurden nun Ihrem **Warenkorb** hinzugefügt.
Nun auf den Button **„Bestellungen anzeigen“** klicken. Im Anschluss gelangen Sie in Ihren Warenkorb.
