# Warenkörbe bei Lieferanten auslösen

**Auslösen von Warenkörben bei Lieferanten**

![speisekarte_bestellen](/img/bestellen_speisekarte/bestellen_mit_speisekarte_7.png)

Sie befinden sich nun in Ihrem Warenkorb. Rezepturen, die Zutaten von verschiedenen Lieferanten enthalten, werden je Lieferant bestellt. D.h. dass
Sie in diesem Fall **mehrere Warenkörbe** haben. Die Bestellungen werden nur je Lieferant zusammengefasst, wenn das Lieferdatum dasselbe ist. Jeder
einzelne Warenkorb enthält alle Zutaten und die Möglichkeit, die Bestellung zu bearbeiten und direkt beim Lieferanten auszulösen. 

![speisekarte_bestellen](/img/bestellen_speisekarte/bestellen_mit_speisekarte_8.png)

Die Mengen, die hier eingetragen sind, sind errechnete Mengen, um Ihre Produktion abdecken zu können. ***Achtung: Sie müssen die Mengen nicht
auf ein volles Gebinde aufrunden. Das macht das System automatisch***. Sollten Sie dennoch die Mengen anpassen wollen, so können Sie über den
Button **„Bearbeiten“** klicken und Ihre **Mengen anpassen**. Ansonsten klicken Sie auf den Button **„Auftrag bestätigen“**, um Ihre Bestellung bei Ihrem
Lieferanten auszulösen.
