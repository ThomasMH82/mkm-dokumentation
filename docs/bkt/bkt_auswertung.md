# BKT-Werte kontrollieren

### BKT-Anzeige einschalten

![speisekarte_bestellen](/img/bkt/bkt_5.png)

So sehen Sie die berechneten BKT-Werte:
1. Klicken Sie auf **"Erweiterte Ansicht"** in der oberen Menüleiste
2. Die BKT-Kosten werden jetzt bei jeder Rezeptur angezeigt
3. Bei kombinierten Rezepturen sehen Sie die Gesamtsumme

### Wochenauswertung ansehen

![speisekarte_bestellen](/img/bkt/bkt_6.png)

Unter **"Kalkulation Speisekarte"** finden Sie eine übersichtliche Zusammenfassung:

Sie können hier auf Rezepturebene den verbleibenden BKT pro Rezeptur sehen.

Im Bereich der Gesamtkalkulation sehen Sie den noch verbleibenden durchschnittlichen auf die Woche kummulierten anteiligen BKT.

:::info Information
Die BKT-Funktion ermöglicht eine präzise Kostenkontrolle und -zuordnung in der Speiseplanung. Die automatische Berechnung kumulierter Werte erleichtert die Kalkulation komplexer Menüs.

Haben Sie an Rezepturen keinen BKT Speisegruppe vergeben, nutzt der MyKitchenManager den Tages BKT.
:::
