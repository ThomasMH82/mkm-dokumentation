# Rezepturen mit BKT verknüpfen

### Normale Rezepturen zuordnen

1. Öffnen Sie die gewünschte Rezeptur
2. Wählen Sie im Dropdown **"Speisegruppe"** die passende Mahlzeit aus:
   - hier im Beispiel: Frühstück, Mittag, Kaffee oder Abendessen
:::tip **Wichtig bei Rezeptur-Kombinationen:**
 Wenn Sie mehrere Rezepturen miteinander kombinieren (kaskadierte Rezepturen), müssen alle zur gleichen Speisegruppe gehören.
:::

![speisekarte_bestellen](/img/bkt/bkt_3.png)


:::warning Achtung
- Pro Unternehmen kann nur ein BKT-Plan gleichzeitig aktiv sein
- Änderungen am aktiven Plan wirken sich sofort auf alle zugeordneten Rezepturen aus
- Bei kaskadierten Rezepturen müssen alle Komponenten derselben Speisegruppe angehören
:::
