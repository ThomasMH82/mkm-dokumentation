# Sonderkost-Rezepturen kennzeichnen

Für spezielle Diät- oder Allergiker-Rezepturen:
1. Aktivieren Sie das Kontrollkästchen **"Sonderkost"**
2. Diese Rezepturen werden dann separat abgerechnet

:::info **Achtung**: 
 Eine Rezeptur kann nicht gleichzeitig einer normalen Speisegruppe **UND** Sonderkost zugeordnet werden.
 :::
