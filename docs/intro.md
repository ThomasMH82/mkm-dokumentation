# Grundlagen

## Dashboard

![dashboard](/img/daschboard_erklaerung.png)

Über die Startseite bzw. über das Dashboard gelangen Sie in **alle Tagesgeschäft relevanten Bereiche**. Mit Klick auf die einzelnen Buttons gelangen
Sie in die jeweiligen Module. Das Dashboard **lässt sich individuell anpassen**. Sollten Sie Zugriff auf mehrere Unternehmen haben, können Sie diese
im Dashboard anzeigen. Hierzu auf das jetzige Unternehmen klicken und das jeweilige Unternehmen auswählen.

## Betrieb wechseln

![betriebwechseln](/img/multicompany_wechsel.png)

Sie haben die Möglichkeit, verschiedene Unternehmen auszuwählen bzw. ein Unternehmen auszuwählen, in welchem nun gearbeitet werden soll
(grau hinterlegt). Für alle Unternehmen, die angehakt sind, lassen sich die Inhalte anzeigen, allerdings nicht bearbeiten. Möchten Sie sich die Inhalte
von mehreren Unternehmen gleichzeigt anzeigen lassen, so müssen Sie mehrere Unternehmen auswählen.
