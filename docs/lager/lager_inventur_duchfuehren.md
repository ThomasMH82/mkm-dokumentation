# Inventur durchführen

**Durchführung einer Inventur**

![lager](/img/lager/lager_5.png)

1. Um eine Inventur durchzuführen, klicken Sie hierfür auf den Button **„Vorgänge“** und im Anschluss auf **„Inventur“**.
2. Im Anschluss erscheint das zweite Fenster. Klicken Sie nun auf **„Anlegen“**.

![lager](/img/lager/lager_6.png)

***<u>Gehen Sie wie folgt vor, um eine neue Inventur zu beginnen:</u>***
1. Geben Sie das Buchungsdatum ein, ab dem die Inventur durchgeführt werden soll
2. Wählen Sie optional bestimmte Lagerorte aus. Wenn kein Lagerort ausgewählt wird, werden alle Produkte in die Inventur einbezogen.
3. Legen Sie fest, wie die gezählten Mengen dargestellt werden sollen:
Standardmäßig werden alle Mengen mit 0 auf der Zählliste angezeigt. Diese Einstellung kann bei Bedarf angepasst werden.
Nach allen gewünschten Einstellungen klicken Sie auf „Inventur starten“, um den Vorgang zu beginnen.

![lager](/img/lager/lager_7.png)

Nun erscheint folgende Maske:
1. Sie können bereits mit Ihrem Tablett durchs Lager gehen und in der Spalte **„Basis-ME (gezählt)“** Ihre **gezählten Mengen** eintragen.
2. Sollten Sie die Inventur selbst nicht ausführen bzw. **ohne Tablett**, sondern mit **ausgedruckter Liste** durchs Lager gehen wollen, drücken Sie auf
den Button **„Zählliste“**. (Wir auf der nächsten erklärt)
3. Schließen Sie Ihre Inventur immer mit dem Button **„Bestandsbuchung durchführen“** ab. Der MKM wird Sie erneut fragen, ob Sie sich sicher sind.
Sofern Sie die Frage bestätigen, ist die Inventur abgeschlossen.

![lager](/img/lager/lager_8.png)

Nachdem Sie auf den Button „Zählliste“ geklickt haben, wird automatisch eine PDF erstellt. Diese können Sie sich nun ausdrucken.
