# Lagermengen anpassen

**Anpassen von Lagermengen**

![lager](/img/lager/lager_1.png)

Sie können entweder über das Dashboard oder über die linke Menüleiste in das Lagermodul gelangen. 

![lager](/img/lager/lager_2.png)

Im Lagermodul angelangt, können Sie sich die Produkte, die im Lager liegen, **ansehen und bearbeiten**. Wenn Sie nur **einzelne Produkte** anpassen
möchten, können Sie das **entsprechende Lager aufklappen** und in die Spalte **„Lagermenge Basis“** oder „Lagermenge EKE“ des jeweiligen Produktes
klicken. Dort können Sie die Mengen ändern. 

![lager](/img/lager/lager_3.png)

Wie im Beispiel zu sehen, können Sie die Menge anpassen und oben links nach der Änderung speichern. Sie können mehrere Produkte anpassen und
dann ebenfalls erst speichern. 

![lager](/img/lager/lager_4.png)

Sie können entweder über das Dashboard oder über die linke Menüleiste in das Lagermodul gelangen.
