# Produkte anlegen

**So lege ich ein Produkt an**

![produkt_anlegen_1](/img/produkte_und_rezepturen/anlegen_produkte_1.png)

1. In das Produktmodul gelangen Sie entweder über die linke Menüleiste oder über das Dashboard (siehe EK-Produkte)
2. Im Produktmodul angekommen, können Sie oben links auf „Anlegen“ klicken, um Ihre Produkte anzulegen.

![produkt_anlegen_2](/img/produkte_und_rezepturen/anlegen_produkte_2.png)

Alle Felder, die **orange** hinterlegt sind, **sind „Pflichtfelder“**. Alle Felder, die **blau** hinterlegt sind, sind **"Kann-Felder"**, die nicht zwingend ausgefüllt
werden müssen (Empfehlung unsererseits). Alle weiteren Felder müssen nicht ausgefüllt werden, wenn Sie diese nicht brauchen.
Klicken Sie hierfür in das Feld und speichern oben links alle getätigten Änderungen. Unter dem Reiter **„Inhalte & Nährwertangaben“** können Sie die
**Nährwerte, Allergene und Inhalts-/ Zusatzstoffe angeben**. Speichern Sie nun das Produkt.

![produkt_anlegen_3](/img/produkte_und_rezepturen/anlegen_produkte_3.png)

Damit eine Zutat bzw. eine Rezeptur bestellt werden kann, ist es zwingend erforderlich, einen Lieferanten für das jeweilige Produkt zu hinterlegen.
1. Öffnen Sie den Reiter **„Einkauf“** im Produkt
2. Wählen Sie einen **Lieferanten** aus. (Diese sollten vor Produktanlage hinterlegt worden sein)
3. Klicken Sie auf **„Eintrag hinzufügen“**, um einen neuen Lieferanteneintrag zu erstellen
4. Tragen Sie anschließend den **Preis** und die **Gültigkeit** des Produkts ein

![produkt_anlegen_4](/img/produkte_und_rezepturen/anlegen_produkte_4.png)

Über den Button **„Schnellanlage“** steht eine vereinfachte Möglichkeit zur **Produktanlage** zur Verfügung.
Es öffnet sich ein Fenster, in dem die **minimal erforderlichen Informationen** für ein neues Produkt abgefragt werden.
Innerhalb dieses Fensters gibt es eine Option **„Lieferant hinzufügen“**, mit der ein **neuer Lieferant** direkt angelegt werden kann.
Zusätzlich besteht die Möglichkeit, eine **zweite Mengeneinheit** (Beispielsweise Gramm für eine Rezeptur) für das Produkt direkt mitanzulegen.
