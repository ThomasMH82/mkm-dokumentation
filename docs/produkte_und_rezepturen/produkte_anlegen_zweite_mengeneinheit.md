# Zweite Mengeneinheit hinterlegen

**Wie lege ich eine zweite Mengeneinheit an**

- Im MyKitchen Manager haben Sie die Möglichkeit, bei einkaufbaren Produkten zusätzliche Einheiten zu hinterlegen. Dies bedeutet,
dass Sie für jedes Produkt nicht nur die Standardeinheit, sondern auch zusätzliche Einheiten definieren können, die bei Bedarf
ebenfalls bestellt oder gelagert werden können.
- Diese Funktion ist besonders nützlich, wenn ein Produkt in unterschiedlichen Mengen oder Verpackungsgrößen verfügbar ist. Sie
können beispielsweise festlegen, dass ein Produkt sowohl als Einzelstück als auch in einer Verpackungseinheit von 10 oder 100
Stück bestellt werden kann. Die zusätzlichen Einheiten können Sie ebenfalls flexibel als Rezeptureinheit (Beispiel: Gramm)
hinterlegen, um die Produktionsplanung besser an die jeweiligen Bedürfnisse anzupassen.
- Sobald Sie das Produkt gespeichert haben, taucht unten die Einkaufseinheit automatisch auf
Auf den folgenden Seiten wird erklärt, wie Sie diese Stück für Stück anlegen können. 

![zweite_mengeneinheit_1](/img/produkte_und_rezepturen/zweite_mengeneinheit_1.png)

Sie gelangen zum Einkaufsmodul entweder über die linke Menüleiste oder direkt über das Dashboard.
Im Einkaufsmodul können Sie das gewünschte Produkt auswählen und öffnen. Scrollen Sie anschließend nach unten und klicken Sie auf den Button
„Mengeneinheit hinzufügen“, um eine weitere Mengeneinheit für das Produkt zu erfassen.

![zweite_mengeneinheit_2](/img/produkte_und_rezepturen/zweite_mengeneinheit_2.png)

Nun öffnet sich folgendes Fenster. Sofern Sie das Produkt in einer zusätzlichen Einkaufseinheit pflegen möchten, nutzen Sie hierfür „Einkauf“.
Möchten Sie stattdessen eine Rezeptureinheit wie bspw. Gramm nutzen, nehmen Sie hierfür „Rezeptur“. Im Anschluss wählen Sie die gewünschte
Einheit aus der Liste aus. Unter „weitere suchen“ finden Sie alle angelegten Einheiten. Sie können allerdings auch nach dem Namen suchen. 

![zweite_mengeneinheit_3](/img/produkte_und_rezepturen/zweite_mengeneinheit_3.png)

Wählen Sie nun „Basiseinheit“ aus. Nun werden Sie gefragt, in welcher Relation die neu hinterlegte Einheit zum EK-Produkt steht. In diesem Fall ist es
Gramm zur Dose. Nun können Sie angeben, wie viel Gramm eine Dose enthält und der Faktor wird im Anschluss errechnet.
Speichern Sie nun Ihre Angaben und im Anschluss gelangen Sie wieder in das Produkt.
