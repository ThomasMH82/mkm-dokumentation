# Rezepturen anlegen

**Anlegen von Rezepturen**

![rezeptur_anlkegen_1](/img/produkte_und_rezepturen/anlegen_rezeptur_1.png)

1. In die Rezepturen gelangen Sie entweder über die linke Menüleiste oder über das Dashboard
2. Im Modul Rezepturen angekommen, können Sie oben links auf „Anlegen“ klicken, um Ihre Rezepturen anzulegen 

![rezeptur_anlkegen_2](/img/produkte_und_rezepturen/anlegen_rezeptur_2.png)

1. Im oberen Teil des Fensters können Sie den Rezepturnamen eingeben, sowie die Zutatenliste bearbeiten (hierzu später mehr).
2. Außerdem sehen Sie hier verschiedene Informationen für Ihre Rezeptur, welche automatisch berechnet werden nachdem die Zutatenliste
ausgefüllt wurde.

![rezeptur_anlkegen_3](/img/produkte_und_rezepturen/anlegen_rezeptur_3.png)

1. Im unteren Teil des Fensters können Sie weitere Informationen zu der Rezeptur pflegen, wie zum Beispiel den Verkaufspreis oder eine zweite
Mengeneinheit (Analog zu Produkten).
2. Die Inhalte und Nährwertangaben werden automatisch aufgrund der gewählten Zutaten berechnet.

## Zutaten hinterlegen

![rezeptur_anlkegen_4](/img/produkte_und_rezepturen/anlegen_rezeptur_4.png)

1. Über den Button „Zutatenliste bearbeiten“ können Sie die einzelnen Komponenten der Rezeptur hinzufügen bzw. bearbeiten.
2. Sofern weitere Mengeneinheiten zu der Zutat gepflegt wurden, tauchen und der „Rez.Einheit“ mehrere Einheiten auf. Das System errechnet
aufgrund des hinterlegten Faktors automatisch die richtige EK-Menge und den Wareneinsatz.
