# Temperaturerfassung Produktionsplanung

**Erfassung von Temperaturen während der Produktion**

![produktionsauftrag](/img/produktion/produktion_5.png)

Über den Button **„Temperaturen erfassen“** hat man die Möglichkeit, die Temperatur der hergestellten Speisen im Produktionsplan zu erfassen.

![produktionsauftrag](/img/produktion/produktion_6.png)

Nun erhält man eine Übersicht über die eingesetzten Produkte und kann für jedes Produkt, die Felder erfassen.
Sobald man alle Temperaturen eingetragen hat, kann man über den Button **„Erfassen“** die Temperaturerfassung abschließen.
