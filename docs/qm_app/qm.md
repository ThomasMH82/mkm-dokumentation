# Übersicht Qualitätsmanagment

**Qualitätsmanagment**

![qm_app](/img/qm_app/qm_app_1.png)

Über das kleine Klemmbrettsymbol in der Navigationsleiste links öffnet man das Qualitätsmanagement Modul

![qm_app](/img/qm_app/qm_app_2.png)

In dieser Übersicht sieht man nun alle neuen Wareneingangsprüfungen, Reklamationen und Temperaturerfassungen. Um sich diese genauer
anzusehen, klickt man nun auf die einzelnen Kacheln.
