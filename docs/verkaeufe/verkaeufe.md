# Verkäufe erfassen

**Erfassen von Ihren Verkäufen**

![verkaeufe](/img/verkaeufe/verkaeufe_1.png)

Um Ihre Verkäufe erfassen zu können, klicken Sie dafür auf die drei Punkte und im Anschluss auf „Verkäufe erfassen“.

![verkaeufe](/img/verkaeufe/verkaeufe_2.png)

1. Tragen Sie in der Spalte **„Verkauft“** Ihre **verkauften Mengen** ein. Verwenden Sie bei Unsicherheit immer die Spalte **„Verkauft 19%“**.
2. Sollten Sie **Ausschuss** haben oder **verkaufte Mengen** einlagern wollen, können Sie diese angeben. Die Differenz wird **automatisch hinterlegt**
***(Ausschuss)***. Sie können die Mengen ebenfalls manuell ändern und hinterlegen.

![verkaeufe](/img/verkaeufe/verkaeufe_3.png)

Im Speiseplan sieht man nun den Status „Verkäufe erfasst.
