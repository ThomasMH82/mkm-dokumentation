# Wareneingang erfassen

**Erfassen eines Wareneingangs**

![wareneingang](/img/wareneingang/wareneingang_1.png)

Nachdem Sie die Bestellung an Ihren Lieferanten versendet haben, befinden Sie sich automatisch im Status „Wareneingang“. Um nachträglich
nochmals den Wareneingang öffnen zu können, müssen Sie die Bestellung erneut öffnen. Klicken Sie im Anschluss auf „Produktzugänge“. 

![wareneingang](/img/wareneingang/wareneingang_2.png)

Sollten **alle Mengen wie bestellt angeliefert** worden sein, können Sie auf den Button **„Bestätigen“** klicken.
Im Anschluss gelangen Sie in die Wareneingangsprüfung. 
***(Optional)***
Sofern **Anpassungen** getätigt werden müssen, können Sie alle notwendigen Mengen je Artikel ändern. Zusätzlich können Sie rechts in der Spalte Ihre
bereits gezählten Artikel abhaken. Das müssen Sie allerdings nicht tun.
