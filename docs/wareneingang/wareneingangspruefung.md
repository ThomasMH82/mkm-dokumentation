# Wareneingangsprüfung durchführen

*Durchführen der Wareneingangsprüfung**


![wareneingang](/img/wareneingang/wareneingang_3.png)

1. Geben Sie zu jedem Punkt an, ob die Ware in Ordnung ist. Sofern in der Bestellung sowohl Frischware als auch TK-Ware vorhanden ist, taucht
dieses Fenster 2x auf. Beide Masken müssen ausgefüllt werden.
2. Im Anschluss bestätigen Sie das Formular. Dieses wird gespeichert und kann von der entsprechenden Abteilung eingesehen werden.
3. Sofern die Ware nicht in Ordnung ist, taucht nach dem Bestätigen der Maske eine weitere Maske auf. In diesem Reklamationsformular geben Sie
die Gründe an, wieso die Ware reklamiert wird. 

![wareneingang](/img/wareneingang/wareneingang_4.png)

Aufgrund dieser Hinterlegung taucht auf der nächsten Seite das Reklamationsformular auf.
